# 🌀 AI-Powered Maze Game (Shadow Maze)

## 📌 Overview

Shadow Maze is an interactive command-line maze game developed in Python.
It allows users to navigate through a maze using natural language commands with intelligent hints and real-time feedback.

---

## 🚀 Features

* 🧭 Maze navigation system
* 🌍 Multilingual commands (English & French)
* 🤖 Intelligent hint system
* ⚡ Real-time interaction
* 🎮 Restart and menu options

---

## 🛠️ Technologies Used

* Python 3
* CLI (Command Line Interface)
* Rich (optional for enhanced UI)

---

## ▶️ How to Run

```bash
python labyrinthe.py
```

---

## 🎯 Objective

This project demonstrates core programming concepts and basic AI-inspired interaction through an interactive maze game.

## 📸 Output
